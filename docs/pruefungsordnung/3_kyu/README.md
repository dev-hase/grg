3. Chudan Soto-Uke / Jodan Uraken-Uchi (Sanchin Dachi)
4. Kake Uke / Jodan/Chudan Ren Zuki (Sanchin Dachi)

6. Shuto Uke / Gedan Kansetsu-Geri (Nekoashi Dachi rechts, links)
